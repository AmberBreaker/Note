1.	显示当前目录的路径
	pwd

2.	查看当前目录下的内容（即所有文件或文件夹）
	ls 参数 目录名
	参数：
		ls			显示在当前目录下的内容
		ls 目录		显示指定目录下的内容
		ls -a		显示当前目录下的内容（包括隐藏文件）
		ls -l		显示当前目录下内容的详细信息
		ls -R		显示当前目录下子目录文件
		ls -w		显示当前目录下中文文件
		ls -la		显示所有文件
		ls -G		带颜色显示

3.	查看和修改环境变量
	3.1	查看环境变量
		env
	3.2	修改并生效环境变量
		1.	打开终端，在终端中输入：
			cd ~ && ls -a && sudo vim .bashrc
		2.	输入你要添加的环境变量
		3.	输入完环境变量后依次按：
			'esc' => 'shift' + ':' => 'wq'
	3.3	最后让你的 shell 配置及时生效：	
		source ~/.bashrc
	3.4	在终端输入 env 查看是否成功
